Falling Balls
Falling Balls is a 2D Android game built using the Android SDK. It uses self-implemented physics and collision dection. The 2D rendering is handled using built in libraries. The AdMob SDK is used to display in-game advertisments. High scores are stored locally in an SQLite database.
3D Pool
Wrote a 3D pool table simulator in OpenGL and C++. Added sound and 3D texturing to make the game more realistic. Created a library to simulate rigid bodies and collissions. Created a library to handle user events. Added sound effects and texturing to make the game more realistic.
Network
Network is the second project in CS61B - Data Structures. Implemented an AI that achieves a 98% win rate against a random player using mini-max game tree search and a heuristic evaluation function. Implemented a variety of data structures including array lists, circular doubly linked lists and hash tables.
2D Basketball
Created a 2D basketball game using OpenGL and C++. Created a library to handle physics, collissions and user events. Added sound and texturing to make the game more realistic. Created an online database using PHP and mySQL that handled player login, registration and high score submission.
